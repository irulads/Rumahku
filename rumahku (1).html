Cara menghubungkan Gmail API
1
Buat project Google Cloud
Buka console.cloud.google.com → New Project → nama "RumahKu" → Create
2
Aktifkan Gmail API
APIs & Services → Library → cari "Gmail API" → Enable
3
Buat OAuth 2.0 Client ID
APIs & Services → Credentials → Create Credentials → OAuth Client ID → Web Application → Authorized JavaScript origins: URL website kamu
4
Buat filter label di Gmail
Di Gmail → Settings → Filters → buat filter: From contains "@bri.co.id OR @klikbca.com OR @mandiri.co.id OR @bni.co.id" → Apply label "Bank"
5
Masukkan Google Client ID
